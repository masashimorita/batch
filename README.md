# Batch

For batch registration

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'batch', :git => 'git://github.com/masashimorita/batch.git', :branch => "master"
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install batch
